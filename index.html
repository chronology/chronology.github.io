Tell stories with your data

The Chronology Project tackles three data-oriented tasks with three powerful tools: Kronos (data collection), Metis (data analysis), and Jia (data visualization). These tools are loosely coupled and stand alone, but can speak with one-another: Metis can query data in Kronos, and Jia can visualize the answers provided by Metis.

Kickstart your data exploration with our getting started guide or jump right in to the source code or docs.

5 minute quickstart

Kronos: Collect

Kronos is a time series storage engine that allows you to store and retrieve timestamped JSON blobs from various backends (e.g., Cassandra, Elastic Search).

Metis: Analyze

Metis is a time series compute engine. It's currently implemented as a thin HTTP endpoint wrapper around the Spark data processing engine.

Jia: Visualize

Jia is a visualization, dashboarding, and data exploration tool. It can speak with Kronos and Metis. It answers questions for humans, rather than for developers.
